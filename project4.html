Mobile Base UR3 Manipulator Robot

Modeling, Forward Kinematics, and Inverse Kinematics Analysis

Project Overview

Designed a mobile base robot with a UR3 manipulator. Validated forward kinematics using the Matlab Peter Corke toolbox and inverse kinematics through Jacobian matrix. Executed pick-and-place tasks utilizing the ROS2 framework within the Gazebo and Rviz environments.

Features

  • Custom mobile base integrated with a UR3 robotic arm
  • Forward kinematics validated using MATLAB Peter Corke's Robotics Toolbox
  • Inverse kinematics computed using the Jacobian matrix
  • Visualization and debugging in RViz
  • ROS 2 integration for control and communication

Tools & Frameworks

  • Ros2
  • Gazebo
  • RViz
  • MATLAB with Peter's Corke's Robotics Toolbox
  • Python (ROS nodes)
  • URDF/Xacro for robot modeling

How It Works

  • Robot Model: The robot consists of a differential-drive mobile base and a UR3 manipulator mounted on top. All joints and links are defined in a URDF/Xacro file.
  • Kinematics:
    • Forward kinematics were tested in MATLAB using Peter Corke’s toolbox.
    • Inverse kinematics were solved analytically and numerically using the Jacobian matrix.
  • Simulation: The robot was launched in Gazebo, where it navigates to a target, manipulates objects, and visualizes tasks in RViz.
  • Task Execution: ROS 2 nodes handle motion control, inverse kinematics solving, and object manipulation.
