Stereo Vision System for 3D Depth Estimation

3D Vision

Tools & Libraries

  • Python, OpenCV, SIFT, RANSAC, Numpy, Matplotlib

Core Concepts

  • Feature Matching, Epipolar Geometry, Rectification, Disparity Mapping, Depth Estimation

Project Summary

  • Developed a complete stereo vision pipeline to compute 3D depth information from stereo image pairs.

  • Implemented a custom feature_detector() function using SIFT to extract and match keypoints between two images.

    Image1
    Image2
  • Filtered keypoint matches using the ratio test and computed the Fundamental Matrix using RANSAC and SVD for robust outlier rejection.

  • Derived the Essential Matrix from camera intrinsics and decomposed it into Rotation and Translation matrices to estimate relative camera pose.

  • Validated camera pose by performing triangulation and selecting the configuration with the highest number of valid 3D points (positive depth).

  • Performed stereo rectification using OpenCV’s stereoRectifyUncalibrated() and verified success via aligned epipolar lines.

  • Designed a custom block-matching algorithm to compute pixel-wise disparity maps by comparing intensity windows across horizontal epilines.

  • Computed per-pixel depth maps using the disparity

  • Visualized normalized disparity and depth maps in both grayscale and colormaps.
